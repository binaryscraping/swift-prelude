# Prelude

A collection of common, Swift, SwiftUI and UIKit helpers to make developing apps easier.

## Installation

```swift
import PackageDescription

let package = Package(
  dependencies: [
    .package(url: "https://github.com/binaryscraping/swift-prelude.git", .branch("main")),
  ]
)
```
